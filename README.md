# Java Calculator

A simple **console-based Calculator** built using Java. This project performs basic arithmetic operations through a menu-driven interface and demonstrates core Java programming concepts.

## Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧮 Remainder (Modulus)
* 📋 Menu-Driven Program
* 🔁 do-while Loop
* 🔀 switch-case
* ⚠️ Exception Handling
* ✅ Input Validation

## Concepts Used

* Java Methods
* Scanner Class
* Conditional Statements (`if`, `switch`)
* Loops (`do-while`)
* Exception Handling (`try-catch-finally`)
* `ArithmeticException`
* `InputMismatchException`

## Technologies

* Java
* JDK 17+ (or any compatible version)

## How to Run

1. Clone the repository

```bash
git clone https://github.com/AmanBarbate/Java-Calculator.git
```

2. Open the project in your IDE (VS Code, IntelliJ IDEA, or Eclipse).

3. Compile the program

```bash
javac Calculator.java
```

4. Run the program

```bash
java Calculator
```
